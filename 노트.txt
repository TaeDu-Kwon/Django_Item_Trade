
아이템 or 상품 거래 및 경매 사이트 

- 배경 - 
아이템 매니아나 바로템 같은 게임 아이템을 거래하는 사이트를 모티브로 하여 제작

- 사용 할 기술 스택 -
Docker, Python, Django, DRF
(React ? 프론트는 잠시 대기)

- 기능 -
회원가입
카카오 로그인 api를 이용하여 로그인 (보류)
상품 CRUD
상품 장바구니
게임 즐겨찾기 (보류)
실시간 채팅
상품 구매 or 경매 시 알림 이메일 
경매 ( 보류 )
카카오페이를 이용하여 코인을 충전 해당 코인을 사용하여 아이템 거래 (카카오패이 결제 api를 이용하여 결재기능)


- 페이지 구성 -
매인페이지 
코인 충전 페이지 ( 카카오페이를 사용하여 코인 충전 )
상품 등록 ( 아이템 이름 및 이미지 가격 내용 등 )
검색 페이지 ( 게임 이름 검색으로 찾기, 리스트는 표 형식으로 이름, 팜삼, 수량 가격, 거래 완료 표시 )
상품 상세 페이지 ( 아이템 이름 및 이미지 가격 내용 ,아이템 등록한 사람과 실시간 채팅 등)
장바구니 ( 아이템 리스트 )
아이템 구매 ( 아이템 내용, 나의 코인 수, 구매 시 코인이 부족한지 체크)
마이페이지 ( 코인 충전 내역, 아이템 구매 내역 등등..)

- app -
users - 회원가입 및 로그인 
game - 게임 등록 
item - 상품 crud 
pay - 코인 충전 및 상품 구매
user info - 장바구니, 코인 현황, 구매내역
